Get Started

Log In and Sign Up

An AI Based User Interface that allows you to get yourself registered as a new user and thereafter Login using the Login credentials.

Adding up Information into the database

Only Admin has the rights of uploading the employee's/student's information along with a sample of 100 pictures that will be taken by the software itself

Training the Model

After a sample of 100 images of an employee/student is taken by the software, the admin needs to click on the "Train the Model" functionality, so that the software can predict the faces of the registered persons accuractely.

Self Attendance through Face ID

By clicking on the "Mark attendance" button, the presence of the registered person is automatically marked. Simultaneously, a CSV file of the same is generated, which can be exported by clicking the "Export CSV" button. In case of any discrepancy, the admin has rights to manually register the attendance and update the same in the records.
